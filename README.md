# Water Polo Stats

Next.js frontend for tracking water polo game statistics.

## Setup
```bash
npm install
cp .env.example .env.local  # Add your BACKEND_URL
npm run dev
```

## Environment Variables

| Variable | Description |
|----------|-------------|
| `BACKEND_URL` | API server URL |
| `NEXT_PUBLIC_API_URL` | Public API URL (same as above) |

## Scripts

- `npm run dev` — Development server
- `npm run build` — Build for production
- `npm start` — Run production build
